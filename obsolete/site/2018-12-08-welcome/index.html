New Website Launched

Posted on December 8, 2018

We’re pleased to announce the launch of our new website in anticipation for a busy year in 2019. This site designed using the “beautiful-jekyll” theme by Dean Attali and consists of a series of flat Markdown files that are rendered as HTML by our host, Github pages.

Our old site and link structure is still available at https://onlineacademiccommunity.uvic.ca/endingsproject/

If you are interested in static sites with Jekyll and Github pages, see this tutorial at the Programming Historian.
